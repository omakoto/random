# VSH

This directory contains a project for a poxix compatible shell written in golang.

- We will eventually add bash custom features too.
- We use readline for interactive user input.


## Directory structure

- src/      Vsh source code
  - cmd/vsh/vsh.go
    - main entry point. This only contains a `main()` function. Actual implementation is under /src.

- scripts/  Various helper scripts
  
- tests/    End-to-end tests

- doc/      Documentation
